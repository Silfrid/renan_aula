Configurações do git

Comandos quando instalamos o git em um computador

  • git config --global user.mail "coloque@seu.email"configura o email cadastrado no Github ou no outro gerenciador de código fonte
  • git config --global user.name "seu nome e sobrenome" configura o email cadastrado no github ou outro gerenciador de codigo fonte

Comandos para salvar alterações

Comandos que usamos para salvar nossas alterações

  • git add atualiza arquivos não salvos ou mesmo, salvar arquivos que não foram adiconados
  • git commit serve para salvar as alterações, eles não seram salvos até você fazer "git commti"

Comandos para inspecionar um repositório

Para inspecionar os diretórios:git status/git tag/git blame/git log

  • git status você poderá ver qual arquivo foi adicionado ou atualizado, no qual não foi feito "git add"
  • git tag são marcadores de pontos especificos do histórico
  • git blame explora um código específico
  • git log pesquisar alterações especificas ou filtre também

Comandos para gerenciar repositórios remotos

Git remote add/-v/origin add/git remote url/remote rename

  • git remote origin fazer um remote
  • git remote add serve para você adicionar um novo remote
  • git remote -v com isso você verifica o seu novo remote
  • git remote url isso o URL do remote de SSH para HTTPS
  • git remote rename serve para renomear o seu remote
